# Biomedical EEG Signal Analyzer (NumPy)

A Python project designed to process and analyze multi-channel EEG (Electroencephalogram) brainwave signals using **NumPy**.

## Features
* **Data Reshaping & Restructuring:** Multi-channel signal organization.
* **Broadcasting Calibration:** Sensor noise and drift removal across channels.
* **Statistical Signal Processing:** Automated detection of signal peaks, mean amplitudes, and variations.
* **Time-Segment Analysis:** Splitting signal arrays chronologically for deep-dive diagnostics.

## 🛠️ Tech Stack
* Python 3.x
* NumPy
